# Job Market Analyzer

A web scraping and data analysis project that collects 1000+ job listings 
from TimesJobs.com and analyzes the Data Analytics job market in India.

# Project Status
In Progress

## What This Project Does
- Scrapes job listings using Scrapy
- Analyzes in-demand skills, top cities, and salary ranges
- Visualizes findings with charts

## Tech Stack
- Python, Scrapy, Pandas, Matplotlib, Seaborn
